Front-End

O curso de Front-End concentra-se no desenvolvimento da parte visual e interativa de sites e aplicativos, ensinando habilidades em HTML, CSS e JavaScript para criar interfaces de usuário atraentes e responsivas.

Data Science

O curso de Data Science é uma jornada para explorar e analisar dados, desenvolvendo habilidades em estatísticas, aprendizado de máquina e visualização para tomar decisões informadas e resolver problemas complexos.

Inteligência Artificial

O curso de Inteligência Artificial oferece uma visão profunda das tecnologias que capacitam as máquinas a aprender, raciocinar e tomar decisões, abrindo as portas para a automação avançada e soluções inteligentes.

DevOps

O curso de DevOps capacita profissionais a otimizar o desenvolvimento e a entrega de software, unindo desenvolvimento e operações para promover a automação, a colaboração e a entrega contínua.

UX/UI

O curso de UX/UI foca na criação de experiências digitais impactantes, combinando design de interface do usuário (UI) e experiência do usuário (UX) para garantir que os produtos sejam intuitivos, funcionais e visualmente atraentes.

Mobile

O curso de Mobile explora o desenvolvimento de aplicativos para dispositivos móveis, abrangendo sistemas operacionais como iOS e Android, capacitando os alunos a criar aplicativos inovadores para smartphones e tablets.
